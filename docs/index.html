GenGo

AI-Powered Text Processing Tool

ローカルまたはリモートLLMを活用した、スマートなテキスト処理アプリケーション。 翻訳、校正、カスタムプロンプト処理を、ショートカットキー一つで実行できます。

ダウンロード View on GitHub

主な機能

GenGoが提供する強力な機能

デュアルLLM対応

ローカルLLM（LM Studio, Ollama）とリモートLLM（OpenAI, Anthropic）の両方に対応。 用途に応じて柔軟に切り替え可能。

複数の事前プロンプト

最大5個の事前プロンプトを登録可能。翻訳、校正、要約など、 よく使う処理をショートカットキーで即座に実行。

カスタマイズ可能なショートカット

各事前プロンプトとオンデマンドプロンプトに、 自由にショートカットキーを割り当て可能。

オンデマンドプロンプト

ショートカット実行時にその場でプロンプトを入力。 柔軟な処理が必要な場合に最適。

スマート翻訳

入力されたテキストの言語を自動判定し、設定した言語間で相互翻訳。 自然で読みやすい翻訳を実現。

多言語UI

日本語・英語に対応したユーザーインターフェース。 10以上の言語間での翻訳をサポート。

プライバシー優先

ローカルLLMを使用すれば、すべての処理をオフラインで完結。 データは外部に送信されません。

リアルタイムストリーミング

LLMの応答をリアルタイムで表示。 処理の進行状況を視覚的に確認できます。

自動適用機能

処理完了後、結果を自動的に適用してウィンドウを閉じる設定が可能。 シームレスなワークフローを実現。

使い方

4ステップで簡単に始められます

1

LLM設定

ローカルLLM（LM Studio, Ollama等）を起動するか、 リモートLLMのAPIキーを設定します。

2

プロンプト設定

最大5個の事前プロンプトを登録し、 それぞれにショートカットキーを割り当てます。

3

テキスト選択

処理したいテキストを任意のアプリケーションで選択します。

4

ショートカット実行

設定したショートカットキーを押すだけで、 即座にAI処理が開始されます。

セットアップガイド

GenGoを始めるための完全ガイド

1

GenGoをインストール

GitHubのリリースページから、お使いのOSに対応したインストーラーをダウンロードしてインストールします。

ヒント: macOSの場合はDMGファイル、Windowsの場合はインストーラー(.exe)をダウンロードしてください。
ダウンロードページへ
GitHubリリースページからインストーラーをダウンロード
2

LM Studioをインストール

ローカルLLMを使用するために、LM Studioをインストールします。LM Studioは無料で使いやすいLLM実行環境です。インストール時には Power UserまたはDeveloperを選択してください。

ヒント: リモートLLM（OpenAI、Anthropicなど）を使用する場合は、この手順をスキップできます。
LM Studio公式サイト
LM Studio公式サイト
3

AIモデルをダウンロード

LM Studioを起動し、検索バーからAIモデルを検索してダウンロードします。

おすすめモデル:
  • gpt-oss-20b - 多言語に最適化された小型高性能モデル
LM StudioでAIモデルを検索・ダウンロード
4

ローカルサーバーを起動

LM Studioの「Local Server」タブを開き、ダウンロードしたモデルを選択して「Start Server」ボタンをクリックします。

注意: デフォルトのポート番号は 1234 です。変更した場合は、GenGoの設定でエンドポイントを調整してください。
LM Studioでローカルサーバーを起動
5

GenGoを設定

GenGoを起動し、トレイアイコンから「Settings」を開きます。プロンプトとショートカットキーを設定します。

基本設定
  • LLMエンドポイント: http://127.0.0.1:1234/v1
  • 事前プロンプト: よく使う処理を最大5個登録
  • ショートカットキー: 各プロンプトに割り当て
GenGoの設定画面
6

使ってみよう！

任意のアプリケーションでテキストを選択し、設定したショートカットキーを押してください。AI処理が開始されます！

完了！
